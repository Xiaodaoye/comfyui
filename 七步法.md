# 小说七步循环法

## 七步法结构

1. 通过情节，写主角有优势/很牛逼，或有潜力发展空间，可很快提升到很牛逼的境界、获得很牛逼的道具、人脉等，从而拥有某种实际上的优势。
2. 反派（本尊或势力成员）出场，明里暗里点出与主角爆发冲突的可能；强调其不知道主角牛逼，制造信息差，同时抬高反派逼格与资源，拉期待。
3. 主角逐步与反派势力（小弟）产生交集并出现摩擦，主角不吃亏；对方盯上主角并在信息差作用下自信“能搞定”。
4. 反派为自身目的推进行动，因各种原因形成的信息差让配角、围观群众等不看好主角，持续拉期待。
5. 冲突最终在戏剧性场景爆发，反派出手，主角爆杀（过程可多次反转）。
6. 主角表现震惊亲友、围观者、反派后台等各层级关系网。
7. 盘点收获与超额收获，继续制造震惊；由此主角获得等级/阶层提升。随后进入下一轮七步法循环。

## 详细拆解

### 第一步：树立主角优势与潜力
- 强调主角当前状态/处境/等级/阶层，展望下一状态/等级/阶层将拥有的力量、资源与可解决的问题；点出主角潜力及本次故事可能达到的实力层次，但当前尚欠条件，需要行动与克服阻碍才能升级，拉起对主角升级的期待。
- 为后续人脉关系网的展示做准备：可引入新配角、写一系列故事的开端征兆/伏笔、引入新地点/区域/副本等。每一步都要展露后续相关信息，让读者期待将要发生的事件，思考主角如何装逼/解决问题。
- 通过一个小事件让主角利用优势/底牌先小试牛刀震惊配角；目的在于简单吹一下当前底牌但不彻底展露，或由人脉中的高层角色侧面吹一下抬逼格，让读者对发挥这张底牌的场景产生期待。

### 第二步：反派出场并制造信息差
- 逐步拉出反派系列（小弟-中层-boss-更高层）所处关系网，让读者明白敌人层级，期待更高层反派也被主角搞定（长期期待），可顺带铺伏笔。
- 强调反派也很屌，抬逼格并突出其资源丰富，为“反派给主角做嫁衣”埋伏笔；反派越牛逼、资源越多，读者越期待主角搞定的场景（反派也可指麻烦/问题/解密等）。
- 一定强调信息差产生的条件与逻辑，避免反派降智；反派不知道主角牛逼的理由要可信，使其嚣张/自信不显降智；通过反派行为心态的描写，拉起“主角用底牌对付反派”的期待。
- 让读者讨厌反派：通过事件或反派的行为、话语凸显其可憎之处，激发“同仇敌忾/渴望解决”的情绪，期待反派/麻烦/问题被解决的场景。

### 第三步：交集与摩擦，主角不吃亏
- 第一次交锋若是主角本人则不能吃亏；若为主角势力的亲近配角与反派冲突，可出现配角受挫；与主角不近的配角可严重受挫吃亏，再将冲突传递到主角处，随后主角做出反应。总体原则：主角无论如何都不能吃亏（苟道或签到流可苟着发育不正面冲突，但发育是为最终打脸）。
- 可借主角优势制造“小爽”但不张扬；底牌不可尽展，以维持信息差。通过底牌（金手指）制造小爽或期待，让读者期待完全发挥底牌/金手指解决问题时的“大爽”。
- 写反派对初次（或多次）冲突后的反应：他们【觉得自己行了】，认为能搞定主角；有时反派不知道主角具体存在，仅在针对主角所在势力或主角所在乎的势力/人。通过反派的描述、独白、台词凸出动机与自信，用信息差让读者期待“真相大白”的那一刻。
- 通过反派方描述，进一步展示其人脉关系网，为后续冲突与下个单元故事反派铺垫，自然过渡，让读者期待更高等级反派被主角干掉、主角获得更多收获。
- 阐明反派与主角持续冲突的动机，符合人设。绝大多数反派动机是“他挡住了我获取利益的路，所以必须干掉他”；主角动机亦需合理。可在冲突爆发前后交代，强调双方必然爆发正面冲突，勾起读者对“主角在冲突时灭掉对方”的期待。

### 第四步：反派推进与“众人不看好”
- 反派行动须在主角以外方面取得某种成效；此时主角与反派势力可继续冲突，也可引而不发，通过主角蓄势与反派嚣张酝酿情绪，但主角仍不能吃亏。此处是为了确认反派逼格确如前文所吹，否则纸老虎不爽。逼格确认即拉起读者期待主角踩它装逼的时刻。
- 反派行动影响在主角与中立人脉关系网中传播，重复强调并展示关系网角色层级，描述他们的反应与人设，为前后对比做准备。【不看好】并非一刀切：主角死忠可100%相信，但友方+中立多数持负面态度。通过这些反应拉起读者对“他们震惊于主角反杀”的期待。
- 理顺重要逻辑：友方与中立方为何【不看好】主角？核心在等级/实力设定：主角的等级、资源、人脉与反派方对比必须在剧情中强调清楚，以产生层次落差感，后续主角出手才能产生爽感。强调等级/阶层差异是标准拉期待手段。
- 若有暧昧对象，此处进行情绪铺垫。依据人设决定其对主角的看法，但最终服务于前后反差以满足“美女倾慕/霸总着迷”等需求。避免死板，结合经历、身世让读者期待两人关系进展。
- 明确主角正在做什么：通常继续提升实力，让实力从“能对付反派”提升到“能随时拿捏反派”，或再拿到更厉害底牌，提升安全感；同时铺设下一个单元故事线索，如新地图、新副本、新区域、新人脉、新人物线索，为下一步行动动机埋伏笔。实力与安全感提升，拉起对本单元对付反派的期待；线索展示拉起读者对下个单元主角装逼的期待。

### 第五步：戏剧性正面冲突爆发
- 即便是小单元故事，也要让最终冲突爆发具有戏剧性：特殊时刻、特殊地点、围观场景等；相关伏笔需提前铺设。提升最终冲突戏剧性的铺垫可置于步骤1–4各处，能提升戏剧性的要素均可用以激发期待。
- 人脉关系网成员在此刻汇聚：可物理围观或线上关注、吃饭、员工大会、宗门比武等；作者角度是为了让他们更方便对主角行为做出【反应】（震惊）。需安排好各就各位，谁先震惊谁后震惊、震惊几轮。当角色齐聚，读者自然期待他们对主角的反应。
- 反派出手前的姿态、心态、信心的描写很重要。此前已强调他们信心满满，此处应通过事件在爆发前与过程中强化其信心；越膨胀、自信，反转到来时戏剧性越强、情绪满足越高。继续多用对比：反派的自信、己方配角的【不看好/消极】反应，对比后续爆杀时他们的反应，形成“胜券在握被打脸”的爽点。
- 主角反杀过程可利用【情绪折线】：先顺利拉升，遇挫折下落，再解决拉升；可一次或多次转折。每次情绪转折配合关系网角色的反应（质疑、不相信、震惊）进行上下行描写。转折目的在放大读者对最终【爆杀】的期待。务必把解决问题的过程写清楚。爆杀后跟上对下一单元的铺垫与伏笔，无论反派人物、事件等，尽量塞；塞不了则在盘点收获时再提。通过对下一个单元故事（甚至久后的事件，如几年后的危机）的线索与伏笔拉起长期或短期期待。

### 第六步：分层震惊与情感推进
- 按前文多次展示的人脉关系网，从低到高逐层写配角反应，阈值逐步提升。关系网构建得好，角色存在感足够，读者会期待所有相关角色都被【震惊】一圈，并在每层级的震惊中获得满足（爽）。当前关系网上层角色的看法与反应要着重写，分量更重、情绪满足更强。
- 若有感情线与暧昧女性角色，安排情节促进后续双方感情增进；着重写暧昧角色对主角看法的变化与反应，以达成情绪满足，描述更进一步的可能或促成事件，拉起感情线期待。
- 震惊可分阶段：当场围观者震惊一轮；消息传播后震惊未围观者；再远一点写旧角色的反应（若能提供情绪价值或后续线索伏笔）；或震惊下一个单元的新角色。当前人脉关系网地位较高者与下个地图新角色的反应都建议写，以有效拉起后续他们对主角评价继续变化的期待。
- 反派/问题解决后，为后续新敌人、新问题拉出可接续的逻辑链条。此前强调反派关系网，此时要把【震惊】传递给反派上司、势力成员，让他们震惊并做出后续反应。若能形成顺畅链条如“打了小的来了老的”，即可拉起主角后续收拾“老的”的期待。

### 第七步：盘点收获与升级
- 强调主角已达成最初保底目的（或读者最初期待的目的）；所有收获（包括超额收获）的本质是让主角在等级/阶层上有实质性提升，否则前功尽弃。盘点收获是重要的情绪满足点，要写得明明白白、逐项讲清楚，避免一笔带过。
- 收获/超额收获可专门写爽点：无意秀出再震惊他人（暧昧对象、顶级大佬等），或用收获顺手解决小麻烦、小问题。关键在带来情绪满足。最好让收获蕴含与后续剧情相关的线索，用以拉起对长短期剧情的期待。
- 专门强调主角在单元故事第一步时的资源/处境/等级/阶层与现在的差别；重点描述由此带来的潜力提升、资源提升、人脉变动，打开继续向上的想象路线。整体地位与资源提升意味着主角拥有更大自由、能做此前期待而做不了的事（或更进一步）；对这些内容的强调能拉起长短期期待。
