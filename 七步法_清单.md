# 七步法检核清单

## 总体原则
- [ ] 信息差是否合理且自洽，反派不降智
- [ ] 主角本人任何正面交锋不吃亏
- [ ] 底牌/金手指不全部暴露，持续拉期待
- [ ] 关系网层级清晰，角色反应可形成“震惊链”
- [ ] 关键节点有伏笔与戏剧性设置（时机/地点/围观）

## 第一步：树立主角优势与潜力
- [ ] 清楚交代主角当前状态/处境/等级/阶层
- [ ] 明确下一等级/阶层目标与达成条件
- [ ] 铺设人脉/新地点/区域/副本/伏笔
- [ ] 通过小事件低调展示优势/底牌但不彻底

## 第二步：反派出场并制造信息差
- [ ] 展开反派关系网（小弟→中层→boss→更高层）
- [ ] 抬高反派逼格与资源，为“做嫁衣”铺垫
- [ ] 信息差成因合理，反派自信不显降智
- [ ] 设计让读者讨厌反派的行为/事件

## 第三步：交集与摩擦，主角不吃亏
- [ ] 首次交锋主角不吃亏（配角受挫有层次）
- [ ] 小爽可有，但底牌保持不完全暴露
- [ ] 反派误判与自信通过独白/台词体现
- [ ] 铺垫更高层反派与后续冲突动机（符合人设）

## 第四步：反派推进与“众人不看好”
- [ ] 反派在主角之外取得阶段性成效
- [ ] 关系网传播“不看好”，区分角色态度差异
- [ ] 等级/资源/人脉对比逻辑清晰，差距明确
- [ ] 暧昧对象情绪铺垫与戏剧性准备
- [ ] 主角蓄势：实力/底牌提升与安全感增强
- [ ] 铺设下一单元线索（新地图/副本/区域/人脉）

## 第五步：戏剧性正面冲突爆发
- [ ] 冲突时机/地点/围观设置提升戏剧性
- [ ] 关系网角色齐聚，安排反应节奏与轮次
- [ ] 反派姿态与信心强化，为打脸对比做准备
- [ ] 运用情绪折线（多次转折）放大爽点
- [ ] 解法过程写清楚，并埋入后续伏笔/线索

## 第六步：分层震惊与情感推进
- [ ] 震惊链从低到高层层推进，上层反应着重描写
- [ ] 感情线顺势推进，暧昧角色反应与变化明确
- [ ] 消息传播引发第二轮/第三轮震惊（旧角色/新角色）
- [ ] 震惊传递至反派上层与势力，安排后续反应

## 第七步：盘点收获与升级
- [ ] 明确保底目的与超额收获逐项盘点
- [ ] 收获带来等级/阶层/资源/人脉的实质性提升
- [ ] 用收获制造次级爽点或解决小麻烦
- [ ] 强调自由度提升与下一步可为之事

## 进入下一轮
- [ ] 自然接入更高层反派或新地图/线索，继续循环
